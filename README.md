# Pipes Game
## What is it?
This is a simple game I made during the past couple days. You have to rotate each pipe, so that each connection is closed and there are no leaks!
The map is generated randomly using the Wave Function Collapse algorithm, and the sprite rotation is done using the three-skew method - because I'm working with a raw pixel buffer!

## How to play?
W/A/S/D - Move Up/Left/Down/Right  
Space - Rotate  
The puzzle is solved when each pipe has its "outputs" connected to other pipe(s). Pipes cannot lead to an empty space or screen edges.
